Списки

Что такое список в Python?

Списки (тип list) — это упорядоченные изменяемые коллекции, которые могут хранить элементы любых типов: числа, строки, другие списки и даже смешанные типы данных. Списки — одна из самых мощных и часто используемых структур данных в Python. Они нужны для хранения последовательностей данных, которые можно изменять: добавлять, удалять или заменять элементы.

Теперь, когда мы знаем, что такое списки, давайте научимся с ними работать. Python предлагает множество удобных операций для манипуляции списками.

Создание списка и доступ к элементам

Списки создаются с помощью квадратных скобок []. Доступ к элементам осуществляется по индексу, начиная с 0. Отрицательные индексы отсчитываются с конца. 

Программный код:

    fruits = ["яблоко", "банан", "апельсин"]
    print(fruits[0])    # Первый элемент (индекс 0)
    print(fruits[1])    # Второй элемент (индекс 1)
    print(fruits[-1])   # Последний элемент (индекс -1)

📝 Пояснение: Список fruits содержит три строковых элемента. Обращение к элементу по индексу 0 даёт "яблоко" — первый элемент. Индекс 1 даёт "банан". Отрицательный индекс -1 означает последний элемент списка — "апельсин". Это удобно, когда нужно получить элемент с конца, не зная точной длины списка. 

Вывод:

              яблоко
              банан
              апельсин

Изменение элементов списка

Так как списки изменяемы, мы можем менять их элементы, обращаясь к ним по индексу и присваивая новое значение. 

Программный код:

    numbers = [1, 2, 3, 4, 5]
    numbers[2] = 10    # Меняем элемент с индексом 2 (третий элемент)
    print(numbers)

📝 Пояснение: Изначально список numbers содержал [1, 2, 3, 4, 5]. Команда numbers[2] = 10 заменяет третий элемент (индекс 2, значение 3) на 10. В результате список становится [1, 2, 10, 4, 5]. Это демонстрирует изменяемость списков — в отличие от строк, которые нельзя изменить после создания. 

Вывод:

              [1, 2, 10, 4, 5]

Добавление элементов (append и insert)

Метод append() добавляет элемент в конец списка, а insert() — на указанную позицию, сдвигая остальные элементы вправо. 

Программный код:

    colors = ["красный", "синий"]
    colors.append("зеленый")           # Добавляем в конец
    colors.insert(1, "желтый")        # Вставляем на позицию 1
    print(colors)

📝 Пояснение: append("зеленый") добавляет новый элемент в самый конец списка. insert(1, "желтый") вставляет элемент "желтый" на позицию с индексом 1 (между "красный" и "синий"). Все элементы, которые были на этой позиции и правее, сдвигаются вправо. append() работает быстрее, так как не требует сдвига элементов. 

Вывод:

              ['красный', 'желтый', 'синий', 'зеленый']

Удаление элементов (remove, pop и del)

remove() удаляет первый найденный элемент по значению, pop() удаляет по индексу и возвращает удалённый элемент, а del удаляет по индексу или срезу. 

Программный код:

    animals = ["кот", "собака", "птица", "рыба"]
    animals.remove("птица")         # Удаляем по значению
    last = animals.pop()            # Удаляем последний элемент и получаем его
    del animals[0]                  # Удаляем первый элемент
    print(animals)
    print("Удаленный элемент:", last)

📝 Пояснение: remove("птица") ищет и удаляет первый элемент со значением "птица". pop() без аргумента удаляет последний элемент и возвращает его (в переменную last попадает "рыба"). del animals[0] удаляет элемент с индексом 0 ("кот"). В итоге остаётся только ["собака"]. Если нужно удалить элемент по индексу, но при этом получить его значение — используйте pop(). Если значение не нужно — подойдёт del. 

Вывод:

              ['собака']
              Удаленный элемент: рыба

Срезы списков

Как и со строками, со списками можно работать с помощью срезов — извлекать подсписки. Синтаксис: [начало:конец:шаг]. 

Программный код:

    numbers = [10, 20, 30, 40, 50, 60]
    print(numbers[1:4])     # Элементы с индекса 1 до 4 (не включая 4)
    print(numbers[::2])     # Каждый второй элемент (шаг 2)
    print(numbers[::-1])    # Обратный порядок (шаг -1)

📝 Пояснение: Срезы создают новый список, не изменяя исходный. [1:4] берёт элементы с индексами 1, 2, 3 (индекс 4 не включается). [::2] — начало и конец не указаны (берётся весь список), шаг 2 означает, что берётся каждый второй элемент. [::-1] — отрицательный шаг идёт в обратном направлении, полностью переворачивая список. Это очень удобно без использования циклов. 

Вывод:

              [20, 30, 40]
              [10, 30, 50]
              [60, 50, 40, 30, 20, 10]

Объединение и повторение списков

Списки можно объединять с помощью оператора + (создаётся новый список) и повторять с помощью * (умножение). 

Программный код:

      list1 = [1, 2, 3]
      list2 = [4, 5, 6]
      combined = list1 + list2   # Склеиваем два списка
      repeated = list1 * 2       # Повторяем список 2 раза
      print("Объединение:", combined)
      print("Повторение:", repeated)

📝 Пояснение: Оператор + создаёт новый список, содержащий все элементы первого списка, затем все элементы второго. Исходные списки не изменяются. Оператор * повторяет список указанное количество раз. Обратите внимание: при умножении списка, содержащего изменяемые объекты, могут возникнуть неожиданные эффекты (вложенные списки будут копироваться по ссылке). 

Вывод:

              Объединение: [1, 2, 3, 4, 5, 6]
              Повторение: [1, 2, 3, 1, 2, 3]

Полезные методы списков: sort, reverse, index, count

Python предоставляет множество удобных методов для работы со списками: сортировка, разворот, поиск индекса и подсчёт вхождений. 

Программный код:

      numbers = [3, 1, 4, 1, 5, 9, 2]
      numbers.sort()                  # Сортировка по возрастанию
      print("После sort():", numbers)
      numbers.reverse()               # Разворот списка
      print("После reverse():", numbers)
      print("Количество единиц:", numbers.count(1))
      print("Индекс числа 5:", numbers.index(5))

📝 Пояснение: sort() изменяет исходный список, сортируя его по возрастанию (можно добавить reverse=True для сортировки по убыванию). reverse() переворачивает порядок элементов. count(1) подсчитывает, сколько раз число 1 встречается в списке. index(5) находит индекс первого вхождения числа 5. Если элемент не найден, index() вызовет ошибку ValueError, поэтому лучше сначала проверить наличие элемента с помощью in. 

Вывод:

              После sort(): [1, 1, 2, 3, 4, 5, 9]
              После reverse(): [9, 5, 4, 3, 2, 1, 1]
              Количество единиц: 2
              Индекс числа 5: 1

Проверка наличия элемента и длина списка

Оператор in проверяет, есть ли элемент в списке, а функция len() возвращает длину списка (количество элементов). 

Программный код:

      fruits = ["яблоко", "банан", "апельсин"]
      print("Есть ли банан?", "банан" in fruits)
      print("Есть ли вишня?", "вишня" in fruits)
      print("Длина списка:", len(fruits))

📝 Пояснение: Оператор in возвращает True, если элемент присутствует в списке, и False в противном случае. Он очень удобен для проверок перед удалением или поиском. len() возвращает количество элементов в списке. Для списка из трёх элементов len() вернёт 3. Важно: len() работает за O(1) — постоянное время, так как длина хранится как отдельное значение. 

Вывод:

              Есть ли банан? True
              Есть ли вишня? False
              Длина списка: 3

Дополнительно: цикл по списку и генераторы списков

Списки удобно перебирать с помощью цикла for, а создавать — с помощью генераторов списков (list comprehensions). 

Программный код:

      fruits = ["яблоко", "банан", "апельсин"]
      for fruit in fruits:
          print(fruit.upper())
      
      squares = [x**2 for x in range(5)]
      print("Квадраты чисел:", squares)

📝 Пояснение: Цикл for перебирает каждый элемент списка по очереди. В примере каждый фрукт выводится в верхнем регистре. Генератор списка [x**2 for x in range(5)] создаёт новый список, где каждый элемент — это квадрат числа от 0 до 4. Генераторы списков — это более краткий и часто более быстрый способ создания списков по сравнению с циклом for и append(). 

Вывод:

              ЯБЛОКО
              БАНАН
              АПЕЛЬСИН
              Квадраты чисел: [0, 1, 4, 9, 16]
