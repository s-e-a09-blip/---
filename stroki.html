Строка

Что такое строка в Python?

Строки (тип str) — это последовательности символов, которые помогают нам хранить и обрабатывать текстовую информацию. Они нужны буквально везде: для вывода сообщений пользователю, хранения имен, адресов, работы с файлами и многого другого!Python предлагает несколько удобных способов создания строк. Вы можете использовать одинарные, двойные или тройные кавычки.

Изменение строк:

Теперь, когда мы умеем создавать строки, давайте научимся с ними работать. Python предлагает множество удобных операций для манипуляции строками.

Сложение строк

Строки можно складывать с помощью оператора +, чтобы создавать новые строки: 

Программный код:

      first_name = "Иван"
      last_name= "Петров"
      full_name= first_name + " " + last_name
      print(full_name)
                    
Вывод:


               Иван Петров

Повторение строк

Программный код:

      star_line = "=" * 10
      print(star_line)
                    
Вывод:


              ==========

Доступ к отдельным символам
Строки в Python – это последовательности символов, и мы можем обращаться к каждому символу по его позиции (индексу). Индексация начинается с 0: 

Программный код:

      world = "Python"
      print(world[-1])
      print(world[0])
                    
Вывод:

              P 
              n

Получение части строки (срезы)
Одна из самых мощных возможностей Python – это срезы (slices). Они позволяют извлекать подстроки, указывая начальный и конечный индексы: 

Программный код:

      world = "Python Programming"
      print(message[0:6])
      print(message[::2])
      print(message[::-1])
                    
Вывод:

              Python
              Pto rgamn 
              gnimmargorP nohtyP

Разделение и объединение строк
Эти методы очень полезны при работе с данными: 

Программный код:

      a = ["Python", "это", "здорово"]
      b" ".join(a)
      print(b)
                    
Вывод:

              Python это здорово 

                    
Программный код:

      a = "Python - язык программирования"
      b= a.split()
      print(b)
                    
Вывод:

        'Python', '-', 'язык', 'программирования'
