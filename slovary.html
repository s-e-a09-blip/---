Словари

Что такое словарь в Python?

Словари (тип dict) — это неупорядоченные изменяемые коллекции, которые хранят данные в формате ключ: значение. В отличие от списков, где доступ к элементу осуществляется по индексу (числу), в словарях доступ идёт по ключу, который может быть любым неизменяемым типом (строки, числа, кортежи). Словари невероятно полезны для быстрого поиска, группировки данных и представления структурированной информации.

Основные операции со словарями:

Теперь, когда мы знаем, что такое словари, давайте научимся с ними работать. Python предлагает множество удобных операций для манипуляции словарями.

Создание словаря и доступ к значениям

Словари создаются с помощью фигурных скобок {} с парами ключ: значение, разделёнными запятыми. Доступ к значениям осуществляется по ключу с помощью квадратных скобок [] или метода get(). 

Программный код:

              person = {
                  "name": "Иван",
                  "age": 25,
                  "city": "Москва"
              }
              print(person["name"])      # Доступ по ключу
              print(person.get("age"))   # Безопасный доступ через get()
              print(person.get("country", "Не указано"))  # Значение по умолчанию

📝 Пояснение: Словарь person хранит информацию о человеке. Обращение person["name"] возвращает значение по ключу "name" — "Иван". Если ключа нет, возникнет ошибка KeyError. Метод get() безопаснее: он возвращает None (или указанное значение по умолчанию), если ключ отсутствует. В третьем примере ключа "country" нет, поэтому возвращается "Не указано". 

Вывод:

                      Иван
                      25
                      Не указано

Добавление и изменение элементов

Словари изменяемы: можно добавлять новые пары или изменять существующие, просто присваивая значение по ключу. 

Программный код:

              person = {"name": "Иван", "age": 25}
              person["city"] = "Москва"        # Добавляем новую пару
              person["age"] = 26              # Изменяем существующее значение
              print(person)

📝 Пояснение: Если ключа "city" в словаре нет, он создаётся с указанным значением. Если ключ "age" уже существует, его значение перезаписывается. Это одна из главных особенностей словарей — они динамически расширяются и позволяют легко обновлять данные. 

Вывод:

                      {'name': 'Иван', 'age': 26, 'city': 'Москва'}

Удаление элементов (pop, popitem и del)

pop() удаляет элемент по ключу и возвращает его значение, popitem() удаляет и возвращает последнюю добавленную пару (в Python 3.7+), del удаляет элемент по ключу. 

Программный код:

              person = {"name": "Иван", "age": 25, "city": "Москва", "job": "инженер"}
              
              age = person.pop("age")           # Удаляем по ключу, получаем значение
              last_item = person.popitem()      # Удаляем последнюю добавленную пару
              del person["city"]                # Удаляем по ключу (значение не возвращается)
              
              print("Удаленный возраст:", age)
              print("Последняя удалённая пара:", last_item)
              print("Оставшийся словарь:", person)

📝 Пояснение: pop("age") удаляет пару с ключом "age" и возвращает значение 25. popitem() удаляет и возвращает последнюю добавленную пару (в Python 3.7+ словари сохраняют порядок вставки). del просто удаляет элемент, не возвращая значение. Если ключа нет, pop() вызовет ошибку, а del — тоже. Используйте pop() с вторым аргументом (например, pop("key", None)), чтобы избежать ошибки. 

Вывод:

                      Удаленный возраст: 25
                      Последняя удалённая пара: ('job', 'инженер')
                      Оставшийся словарь: {'name': 'Иван'}

Перебор словаря (keys, values, items)

Для перебора словаря используются методы keys() (ключи), values() (значения) и items() (пары ключ-значение). 

Программный код:

              person = {"name": "Иван", "age": 25, "city": "Москва"}
              
              print("Ключи:")
              for key in person.keys():
                  print(f"  {key}")
              
              print("Значения:")
              for value in person.values():
                  print(f"  {value}")
              
              print("Пары ключ-значение:")
              for key, value in person.items():
                  print(f"  {key}: {value}")

📝 Пояснение: keys() возвращает все ключи, values() — все значения, items() — кортежи (ключ, значение). Метод items() особенно удобен, так как позволяет в цикле сразу получить и ключ, и значение. В Python 3.7+ порядок перебора соответствует порядку добавления элементов в словарь. 

Вывод:

                      Ключи:
                        name
                        age
                        city
                      Значения:
                        Иван
                        25
                        Москва
                      Пары ключ-значение:
                        name: Иван
                        age: 25
                        city: Москва

Проверка наличия ключа (in)

Оператор in проверяет, есть ли ключ в словаре. Это очень быстрая операция (O(1) — постоянное время). 

Программный код:

              person = {"name": "Иван", "age": 25, "city": "Москва"}
              
              if "name" in person:
                  print("Ключ 'name' существует!")
                  print(f"Его значение: {person['name']}")
              
              if "country" not in person:
                  print("Ключа 'country' нет в словаре")

📝 Пояснение: Оператор in проверяет наличие ключа, а не значения. Это очень эффективно, так как словари используют хэш-таблицы для мгновенного поиска. Всегда используйте in перед прямым доступом person["key"], если не уверены в существовании ключа, чтобы избежать ошибки KeyError. 

Вывод:

                      Ключ 'name' существует!
                      Его значение: Иван
                      Ключа 'country' нет в словаре

Методы update и setdefault

update() объединяет словари (добавляет или обновляет пары), setdefault() возвращает значение по ключу, а если ключа нет — создаёт его с указанным значением. 

Программный код:

              person = {"name": "Иван", "age": 25}
              new_data = {"age": 26, "city": "Москва"}
              
              person.update(new_data)      # Обновляем/добавляем данные
              print("После update():", person)
              
              country = person.setdefault("country", "Россия")
              print("Значение country:", country)
              print("После setdefault():", person)

📝 Пояснение: update() добавляет все пары из new_data в person. Ключ "age" обновляется, ключ "city" добавляется. setdefault("country", "Россия") проверяет: если ключа "country" нет, он создаётся со значением "Россия", и это значение возвращается. Если ключ уже существует, метод просто вернёт его значение, ничего не изменяя. 

Вывод:

                      После update(): {'name': 'Иван', 'age': 26, 'city': 'Москва'}
                      Значение country: Россия
                      После setdefault(): {'name': 'Иван', 'age': 26, 'city': 'Москва', 'country': 'Россия'}

Длина словаря (len) и очистка (clear)

len() возвращает количество пар ключ-значение в словаре, clear() удаляет все элементы. 

Программный код:

              person = {"name": "Иван", "age": 25, "city": "Москва"}
              print(f"Количество элементов: {len(person)}")
              
              person.clear()
              print(f"После очистки: {person}")
              print(f"Длина после очистки: {len(person)}")

📝 Пояснение: len() возвращает количество пар в словаре. clear() удаляет все элементы, превращая словарь в пустой {}. После очистки len() вернёт 0. clear() полезен, когда нужно переиспользовать переменную словаря, но удалить все старые данные. 

Вывод:

                      Количество элементов: 3
                      После очистки: {}
                      Длина после очистки: 0

Вложенные словари

Словари могут содержать другие словари, списки и любые другие структуры данных, создавая сложные иерархические структуры. 

Программный код:

              students = {
                  "student1": {
                      "name": "Иван",
                      "grades": [5, 4, 5, 3]
                  },
                  "student2": {
                      "name": "Мария",
                      "grades": [5, 5, 5, 4]
                  }
              }
              
              print(students["student1"]["name"])
              print(students["student2"]["grades"])
              avg_grade = sum(students["student2"]["grades"]) / len(students["student2"]["grades"])
              print(f"Средний балл Марии: {avg_grade}")

📝 Пояснение: Вложенные словари позволяют создавать сложные структуры данных. Здесь словарь students содержит двух студентов, каждый из которых — отдельный словарь с именем и списком оценок. Доступ к вложенным данным осуществляется последовательным указанием ключей. Это очень похоже на работу с JSON — популярным форматом обмена данными. 

Вывод:

                      Иван
                      [5, 5, 5, 4]
                      Средний балл Марии: 4.75

Генераторы словарей (dict comprehensions)

Как и для списков, для словарей существуют генераторы — компактный способ создания словарей из итерируемых объектов. 

Программный код:

              squares = {x: x**2 for x in range(5)}
              print("Квадраты чисел:", squares)
              
              fruits = ["яблоко", "банан", "апельсин"]
              fruit_lengths = {fruit: len(fruit) for fruit in fruits}
              print("Длина слов:", fruit_lengths)
              
              even_squares = {x: x**2 for x in range(10) if x % 2 == 0}
              print("Квадраты чётных чисел:", even_squares)

📝 Пояснение: Генераторы словарей имеют синтаксис {ключ: значение for элемент in последовательность}. В первом примере числа от 0 до 4 становятся ключами, а их квадраты — значениями. Во втором — фрукты становятся ключами, а длины названий — значениями. В третьем — добавляется условие if, чтобы включить только чётные числа. Генераторы делают код короче и часто более читаемым. 

Вывод:

                      Квадраты чисел: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
                      Длина слов: {'яблоко': 6, 'банан': 5, 'апельсин': 8}
                      Квадраты чётных чисел: {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}
