Зачем это надо

Любое мобильное или веб-приложение должно гибко реагировать на действия пользователя. Пользователи постоянно принимают решения: нажимают кнопки, выбирают настройки, вводят данные. Задача программы — отвечать на эти действия, динамически подстраивая интерфейс и свою логику под выбор человека. Этот принцип делает приложение «умным» и интуитивно понятным.

Основу для такого динамического поведения в программировании закладывают условные конструкции и циклы.Эти структуры не только делают программу адаптивной, но и помогают оптимизировать её работу, делая код эффективнее. Для Python-разработчика уверенное владение условиями и циклами — это обязательный базовый навык. Они являются фундаментом для изучения более сложных тем, таких как функции, модули и объектно-ориентированное программирование.

Что такое арифметические операторы в Python?

Чтобы использовать условия и циклы на практике, необходимо понимать операторы сравнения.

Как следует из названия, эти операторы сравнивают два значения (операнда) и возвращают результат в виде логического значения — True (Истина) или False (Ложь). Именно этот результат (True или False) определяет, выполнится ли блок кода в условии или сколько раз отработает цикл.

Уверенное владение условиями, циклами и операторами сравнения — обязательный базовый навык для Python-разработчика. Этот фундамент критически важен для перехода к более сложным темам, таким как функции, модули и объектно-ориентированное программирование.

Основные типы арифметических операторов в Python

Знаки меньше и больше (< >)

Самый простой и понятный оператор, но даже тут есть нюансы:

Оператор «меньше» проверяет, меньше ли значение левого операнда, чем значение правого операнда. Если присмотреться к двум приведенным примерам, в первом случае выводится True, потому что 64 меньше 83, а во втором примере выводится False — ведь 73 точно не меньше 45. Аналогичным образом оператор «меньше» может использоваться для сравнения чисел с плавающей точкой, строк и многого другого. 

Программный код:

              x = 64 < 83 

              y = 73 < 45 
 
              print(x) 

              print(y)
            
Вывод:

              True 
 
              False

Сравнение строк

Программный код:

              x = 'Sample' < 'sample' 

              print(x)
            
Вывод:

              True

В этом примере выводится логическое значение True, потому что ASCII-код буквы нижнего регистра обычно больше ASCII-кодов букв верхнего регистра.

Все также но наоборот работает оператор ">"

Приравнивание (==)

Также один из простых операторов.

Оператор «равно» проверяет, равны ли значения двух операндов — левого и правого. Если значения равны, возвращается логическое значение True, а если нет — значение False. 

Программный код:

              print(64 == 64)

              print(43 == 42)
            
Вывод:

              True 
 
              False

При выполнении программы в IDE или IDLE интерпретатор вычисляет разность двух операндов и сохраняет ее в переменной result.Оператор «неравно» обозначается так: !=

Теперь вы знаете, как работают операторы сравнения, и мы можем перейти к операторам управления, знать которые разработчикам просто необходимо. Программисты используют их даже в относительно простом коде.

Условные операторы зависят от принятия решений при выполнении конкретных операций. Если условие не выполняется, соответствующий блок условной логики пропускается.

Для выбора между двумя блоками в Python существует базовая команда if/else.

if условие:
ffffffвыполняемая команда
else:
ffffffвыполняемая команда 

Программный код:

      number = 43
      if number % 3 == 0:
        print("This number is divided by 3")
      else:
        print("This number is not divisible by 3")
          
Вывод:

            This number is not divisible by 3

Как работает код

  • Сначала необходимо определить переменную, которая будет хранить данные, проверяемые условием if/else.
  • Код, который выполняется после команды if (else), должен иметь отступ (четыре знака «пробел»)
  • В более сложных программах используются средства ввода (input) для получения данных непосредственно от пользователя
  • После того как переменная будет сохранена, интерпретатор обрабатывает условие блока if.
  • Интерпретатор Python выполняет операцию вычисления остатка и определяет, кратно ли 3 проверяемое число.
  • Если число кратно 3, выполняется блок, расположенный под ключевым словом if.
  • Так как в данном случае условие ложно, интерпретатор пропускает блок if и выполняет код блока else. В результате выводится то значение, которое вы видите

В следующем примере выполняется код блока if.

Программный код:

      number = 48
      if number % 3 == 0:
        print("This number is divided by 3")
      else:
        print("This number is not divisible by 3")
          
Вывод:

            This number is divided by 3

Как работает код

Так как условие истинно, выполняется команда print блока if,а блок else пропускается интерпретатором

Чтобы сделать серию проверок более эффективной, можно добавить несколько проверок выполнения условий. 

Программный код:

    number = 45
    if number % 3 == 0:
      print("This number is divided by 3")
    elif number % 4 == 0:
      print("This number is divisible by 4")
    else:
      print("This number is not divisible by 3 and 4")
          
Вывод:

            This number is divisible by 3

Как работает код

В этом примере интерпретатор Python должен проверить три условия. Определив, что первое условие истинно, интерпретатор Python выводит его и пропускает два других условия
