Кортежи

Что такое кортеж в Python?

Кортежи (тип tuple) — это упорядоченные неизменяемые коллекции, которые могут хранить элементы любых типов. Кортежи очень похожи на списки, но их главное отличие — они не могут быть изменены после создания. Кортежи создаются с помощью круглых скобок () и часто используются для хранения связанных данных, которые не должны меняться (например, координаты точки, RGB-цвета, даты).

💡 Важно: Неизменяемость кортежей делает их безопасными для использования в качестве ключей словаря и более эффективными по памяти по сравнению со списками.

Основные операции с кортежами:

Теперь, когда мы знаем, что такое кортежи, давайте научимся с ними работать. Python предлагает множество операций для работы с кортежами, похожих на операции со списками.

Создание кортежа

Кортежи создаются с помощью круглых скобок (). Для создания кортежа из одного элемента нужна запятая после элемента. 

Программный код:

      empty_tuple = ()                      # Пустой кортеж
      point = (10, 20)                    # Кортеж из двух элементов
      colors = ("красный", "зеленый", "синий")  # Кортеж из трёх элементов
      single_element = (5,)                # Кортеж из одного элемента (важно!)
      not_tuple = (5)                     # Это просто число 5, а не кортеж!
      
      print("Пустой кортеж:", empty_tuple)
      print("Точка:", point)
      print("Цвета:", colors)
      print("Кортеж из одного элемента:", single_element)
      print("Тип (5):", type(not_tuple))

📝 Пояснение: Кортежи создаются с помощью круглых скобок. Особое внимание обратите на создание кортежа из одного элемента — нужна запятая после элемента (5,). Без запятой (5) — это просто число int. Пустой кортеж создаётся как (). Кортежи могут содержать элементы разных типов. 

Вывод:

              Пустой кортеж: ()
              Точка: (10, 20)
              Цвета: ('красный', 'зеленый', 'синий')
              Кортеж из одного элемента: (5,)
              Тип (5): <class 'int'>

Доступ к элементам кортежа

Доступ к элементам кортежа осуществляется по индексу (начиная с 0), как в списках. Отрицательные индексы отсчитываются с конца. 

Программный код:

      point = (10, 20, 30, 40, 50)
      print("Первый элемент (индекс 0):", point[0])
      print("Второй элемент (индекс 1):", point[1])
      print("Последний элемент (индекс -1):", point[-1])
      print("Предпоследний элемент (индекс -2):", point[-2])

📝 Пояснение: Как и в списках, доступ к элементам кортежа происходит по индексу в квадратных скобках. Индексация начинается с 0. Отрицательные индексы позволяют обращаться к элементам с конца: -1 — последний элемент, -2 — предпоследний и так далее. 

Вывод:

              Первый элемент (индекс 0): 10
              Второй элемент (индекс 1): 20
              Последний элемент (индекс -1): 50
              Предпоследний элемент (индекс -2): 40

Неизменяемость кортежей

Кортежи нельзя изменять после создания — это их главное отличие от списков. 

Программный код:

      my_tuple = (1, 2, 3, 4, 5)
      print("Исходный кортеж:", my_tuple)
      
      # Попробуем изменить элемент - это вызовет ошибку!
      # my_tuple[2] = 10  # TypeError: 'tuple' object does not support item assignment
      
      # Попробуем добавить элемент - тоже ошибка
      # my_tuple.append(6)  # AttributeError: 'tuple' object has no attribute 'append'
      
      print("Кортежи нельзя изменять после создания")

📝 Пояснение: Кортежи — неизменяемые последовательности. Любая попытка изменить существующий кортеж (изменить элемент, добавить новый или удалить) вызовет ошибку. Это свойство делает кортежи безопасными для хранения константных данных и позволяет использовать их в качестве ключей словаря. 

Вывод:

              Исходный кортеж: (1, 2, 3, 4, 5)
              Кортежи нельзя изменять после создания

Срезы кортежей

Как и со списками, с кортежами можно работать с помощью срезов, которые возвращают новый кортеж. 

Программный код:

      numbers = (10, 20, 30, 40, 50, 60, 70)
      print("Весь кортеж:", numbers)
      print("Элементы с 1 по 4 (не включая 4):", numbers[1:4])
      print("Каждый второй элемент:", numbers[::2])
      print("Кортеж в обратном порядке:", numbers[::-1])
      print("Первые 3 элемента:", numbers[:3])
      print("Последние 3 элемента:", numbers[-3:])

📝 Пояснение: Срезы кортежей работают так же, как и срезы списков. Они создают новый кортеж, содержащий указанные элементы. Синтаксис: [начало:конец:шаг]. Начало включается, конец — не включается. Отрицательные индексы работают как обычно. Шаг может быть отрицательным для обращения порядка. 

Вывод:

              Весь кортеж: (10, 20, 30, 40, 50, 60, 70)
              Элементы с 1 по 4 (не включая 4): (20, 30, 40)
              Каждый второй элемент: (10, 30, 50, 70)
              Кортеж в обратном порядке: (70, 60, 50, 40, 30, 20, 10)
              Первые 3 элемента: (10, 20, 30)
              Последние 3 элемента: (50, 60, 70)

Объединение и повторение кортежей

Кортежи можно объединять с помощью оператора + и повторять с помощью *. 

Программный код:

      tuple1 = (1, 2, 3)
      tuple2 = (4, 5, 6)
      
      combined = tuple1 + tuple2
      repeated = tuple1 * 3
      
      print("Объединение:", combined)
      print("Повторение:", repeated)

📝 Пояснение: Оператор + создаёт новый кортеж, объединяя элементы двух кортежей. Оператор * повторяет кортеж указанное количество раз. Обратите внимание: исходные кортежи не изменяются, а создаются новые. Это соответствует принципу неизменяемости кортежей. 

Вывод:

              Объединение: (1, 2, 3, 4, 5, 6)
              Повторение: (1, 2, 3, 1, 2, 3, 1, 2, 3)

Методы кортежей (count и index)

У кортежей есть два основных метода: count() — подсчитывает количество вхождений элемента, и index() — находит индекс первого вхождения элемента. 

Программный код:

      numbers = (1, 2, 3, 1, 4, 1, 5, 1)
      
      count_ones = numbers.count(1)
      first_index = numbers.index(3)
      first_index_one = numbers.index(1)
      
      print("Количество единиц:", count_ones)
      print("Индекс числа 3:", first_index)
      print("Индекс первой единицы:", first_index_one)
      
      # Поиск с указанием диапазона
      index_in_range = numbers.index(1, 2, 6)  # Ищем 1 с индекса 2 до 6
      print("Индекс единицы в диапазоне [2:6]:", index_in_range)

📝 Пояснение: count() возвращает количество раз, которое элемент встречается в кортеже. index() возвращает индекс первого вхождения элемента. Можно указать начальный и конечный индексы для поиска в диапазоне. Если элемент не найден, index() вызовет ошибку ValueError. 

Вывод:

              Количество единиц: 4
              Индекс числа 3: 2
              Индекс первой единицы: 0
              Индекс единицы в диапазоне [2:6]: 3

Длина кортежа и проверка наличия элемента

Функция len() возвращает длину кортежа, оператор in проверяет наличие элемента. 

Программный код:

      colors = ("красный", "зеленый", "синий", "желтый")
      
      print("Длина кортежа:", len(colors))
      print("Есть ли 'зеленый'?", "зеленый" in colors)
      print("Есть ли 'фиолетовый'?", "фиолетовый" in colors)
      print("Нет ли 'черного'?", "черный" not in colors)

📝 Пояснение: len() возвращает количество элементов в кортеже. Оператор in проверяет, содержится ли элемент в кортеже, и возвращает True или False. Оператор not in проверяет отсутствие элемента. Эти операции работают очень быстро. 

Вывод:

              Длина кортежа: 4
              Есть ли 'зеленый'? True
              Есть ли 'фиолетовый'? False
              Нет ли 'черного'? True

Распаковка кортежей (tuple unpacking)

Распаковка — это присвоение элементов кортежа отдельным переменным. Это очень удобная возможность Python. 

Программный код:

      point = (10, 20)
      x, y = point
      print(f"x = {x}, y = {y}")
      
      person = ("Иван", 25, "Москва")
      name, age, city = person
      print(f"Имя: {name}, Возраст: {age}, Город: {city}")
      
      # Распаковка с использованием * для сбора остатка
      numbers = (1, 2, 3, 4, 5)
      first, *rest = numbers
      print(f"Первый: {first}, Остальные: {rest}")
      
      *beginning, last = numbers
      print(f"Начало: {beginning}, Последний: {last}")

📝 Пояснение: Распаковка позволяет присвоить элементы кортежа переменным в одной строке. Количество переменных должно совпадать с количеством элементов (если не используется *). Звёздочка * собирает оставшиеся элементы в список. Это удобно, когда нужно получить только некоторые элементы кортежа. 

Вывод:

              x = 10, y = 20
              Имя: Иван, Возраст: 25, Город: Москва
              Первый: 1, Остальные: [2, 3, 4, 5]
              Начало: [1, 2, 3, 4], Последний: 5

Вложенные кортежи

Кортежи могут содержать другие кортежи, создавая иерархические структуры данных. 

Программный код:

      matrix = ((1, 2, 3), (4, 5, 6), (7, 8, 9))
      
      print("Вся матрица:", matrix)
      print("Вторая строка:", matrix[1])
      print("Элемент во 2-й строке, 3-м столбце:", matrix[1][2])
      
      # Перебор вложенного кортежа
      for row in matrix:
          for element in row:
              print(element, end=" ")
          print()

📝 Пояснение: Вложенные кортежи позволяют создавать многомерные структуры. Доступ к элементам осуществляется последовательным указанием индексов: matrix[1][2] — сначала получаем вторую строку (индекс 1), затем третий элемент в этой строке (индекс 2). Вложенные кортежи полностью сохраняют свойство неизменяемости. 

Вывод:

              Вся матрица: ((1, 2, 3), (4, 5, 6), (7, 8, 9))
              Вторая строка: (4, 5, 6)
              Элемент во 2-й строке, 3-м столбце: 6
              1 2 3
              4 5 6
              7 8 9

Преобразование между списками и кортежами

Можно легко преобразовывать списки в кортежи и наоборот с помощью функций tuple() и list(). 

Программный код:

      my_list = [1, 2, 3, 4, 5]
      my_tuple = tuple(my_list)
      print("Список:", my_list)
      print("Кортеж из списка:", my_tuple)
      
      another_tuple = (10, 20, 30, 40)
      another_list = list(another_tuple)
      print("Кортеж:", another_tuple)
      print("Список из кортежа:", another_list)
      
      # Можно изменять список, полученный из кортежа
      another_list.append(50)
      print("Изменённый список:", another_list)

📝 Пояснение: tuple() создаёт новый кортеж из любой итерируемой последовательности (списка, строки, множества). list() создаёт новый список из кортежа. Это удобно, когда нужно временно изменить данные: преобразовать кортеж в список, изменить, а затем преобразовать обратно в кортеж. 

Вывод:

              Список: [1, 2, 3, 4, 5]
              Кортеж из списка: (1, 2, 3, 4, 5)
              Кортеж: (10, 20, 30, 40)
              Список из кортежа: [10, 20, 30, 40]
              Изменённый список: [10, 20, 30, 40, 50]

Кортежи как ключи словаря

Благодаря неизменяемости, кортежи могут использоваться в качестве ключей словаря (в отличие от списков). 

Программный код:

      # Словарь с кортежами в качестве ключей
      coordinates = {
          (0, 0): "Начало координат",
          (1, 0): "Точка на оси X",
          (0, 1): "Точка на оси Y",
          (1, 1): "Единичный квадрат"
      }
      
      print(coordinates[(0, 0)])
      print(coordinates[(1, 1)])
      
      # Проверка наличия ключа-кортежа
      if (2, 3) in coordinates:
          print("Есть точка (2, 3)")
      else:
          print("Точки (2, 3) нет в словаре")
      
      # Список НЕ может быть ключом словаря
      # bad_dict = {[1, 2]: "список"}  # TypeError: unhashable type: 'list'

📝 Пояснение: Кортежи являются хэшируемыми (hashable), потому что они неизменяемы. Это позволяет использовать их в качестве ключей словаря. Списки изменяемы, поэтому они не могут быть ключами. Это одно из главных преимуществ кортежей перед списками. 

Вывод:

              Начало координат
              Единичный квадрат
              Точки (2, 3) нет в словаре
